# 02 — 输出格式说明

> `inp-tool post` / GUI / Python API 的 4 类输出文件,字段含义。

---

## 1. Excel 报告(`ForceReport.xlsx`)

`write_excel` / `inp-tool post report` / GUI "Excel" 按钮的产物。

### 1.1 文件结构

- 单 sheet,默认名 `Forces`(`--sheet-name` 不当前暴露;用 API `write_excel(report, path, sheet_name="...")`)
- 第 1 行 header(Times New Roman 10pt 粗体居中)
- 第 2+ 行 data(一 case 一行,Times New Roman 10pt 右对齐)
- 28 列,列宽自适应

### 1.2 28 列详解

| 列 | 名称 | 单位 | 含义 |
|---|---|---|---|
| 1 | Case | — | 算例目录名(不含路径)|
| 2 | Ma | — | 来流马赫数(从 `aero_ma` 读)|
| 3 | H(km) | km | 来流 geopotential 高度(`aero_altid`)|
| 4 | Alpha(deg) | 度 | 攻角(intyp=3 时从 uvw 反推)|
| 5 | Beta(deg) | 度 | 侧滑角(同上)|
| 6 | Fx(N) | N | x 方向总力(已平移到 (Xref,Yref,Zref))|
| 7 | Fy(N) | N | y 方向总力 |
| 8 | Fz(N) | N | z 方向总力 |
| 9 | Mx(N·m) | N·m | x 方向力矩(已平移)|
| 10 | My(N·m) | N·m | y 方向力矩(已平移)|
| 11 | Mz(N·m) | N·m | z 方向力矩(已平移)|
| 12 | D(N) | N | 阻力 = Fx·cos α + Fz·sin α |
| 13 | L(N) | N | 升力 = -Fx·sin α + Fz·cos α |
| 14 | CD | — | 阻力系数 = Fx / (Q·Sref) |
| 15 | CY | — | 侧向力系数 = Fy / (Q·Sref) |
| 16 | CL | — | 升力系数 = Fz / (Q·Sref) |
| 17 | Cmx | — | x 力矩系数 = Mx / (Q·Sref·Lref) |
| 18 | Cmy | — | y 力矩系数 = My / (Q·Sref·Lref) |
| 19 | Cmz | — | z 力矩系数 = Mz / (Q·Sref·Lref) |
| 20 | L/D | — | 升阻比;\|D\|<1e-10 时为 0 |
| 21 | Xcp(m) | m | 压心位置 = Xref - My / Fz;\|Fz\|<1e-10 时为 0 |
| 22 | Xcg(m) | m | 重心位置(用户填,不参与计算)|
| 23 | Sref(m2) | m² | 参考面积 |
| 24 | Lref(m) | m | 参考长度 |
| 25 | Q(Pa) | Pa | 来流动压 = 0.5·ρ·V² |
| 26 | Re(1/m) | /m | 单位 Reynolds 数 = ρ·V/μ |
| 27 | P(Pa) | Pa | 来流压强(`aero_pres`)|
| 28 | T(K) | K | 来流温度(`aero_temp`)|

### 1.3 注意事项

- **力矩中心已平移**:列 9-11 是平移到 `(Xref, Yref, Zref)` 后的值。原始 CFD++ 输出取矩中心是 `(0, 0, 0)`,这里**不是**原始值
- **Q·Sref < 1e-12 时所有系数为 0**:防 0 除
- **Re 用理想气体律**:`Re = P·V / (R·T·μ)`,R=287.0531 J/(kg·K)

## 2. 收敛报告(`convergence_report.txt`)

`format_convergence_report` / `inp-tool post convergence` / GUI "收敛分析" 按钮的产物。

### 2.1 完整示例

```
======================================================================
  CFD++ 气动力收敛性分析报告
======================================================================

收敛判定标准: 各分量变异系数 CV < 0.10%
收敛窗口: 末 10% 迭代步(至少 100 步)

======================================================================
  算例: case_01
======================================================================

  总迭代步: 1500
  收敛窗口: 末 150 步

  -------------------------------------------------------
  分量    CV(%)            收敛状态
  -------------------------------------------------------
  Fx      0.0123           收敛
  Fy      0.0089           收敛
  Fz      0.0456           收敛
  Mx      0.0876           收敛
  My      0.1234           未收敛
  Mz      0.0234           收敛
  -------------------------------------------------------
  综合判定: 未全部收敛

======================================================================
  算例: case_02
======================================================================

  数据量不足,无法判断收敛性

======================================================================
  报告生成时间: 2026-06-14 19:30:00
======================================================================
```

### 2.2 字段含义

- **总迭代步**:`mcfd.info1` 中 `nt` 标记总数
- **收敛窗口**:实际用于计算 CV 的步数 = `max(min_window, n_total · fraction)`
- **CV(%)**:`std(ddof=1) / |mean|` × 100%
- **收敛状态**:`CV < threshold` 即 "收敛"
- **综合判定**:6 个分量都收敛 = "全部收敛";否则 "未全部收敛"

### 2.3 何时 "数据不足"

- `n_total < min_window`(默认 100 步)→ 返回 None,报告说"数据量不足"

## 3. 收敛曲线 png(`convergence_plot.png`)

`save_convergence_plot` / `inp-tool post plot` / GUI "收敛图" 按钮的产物。

### 3.1 布局

```
   ┌──────────────┬──────────────┬──────────────┐
   │     Fx       │     Fy       │     Fz       │
   ├──────────────┼──────────────┼──────────────┤
   │     Mx       │     My       │     Mz       │
   └──────────────┴──────────────┴──────────────┘
```

- 2×3 subplot
- 14×9 英寸,150 DPI
- 横轴:nstep(迭代步)
- 纵轴:对应分量值(N 或 N·m)
- 多 case 时显示 legend(loc="best",8pt)
- 单 case 时不显示 legend(避免遮挡)
- 网格半透明 `alpha=0.3`

### 3.2 怎么看

- **收敛**:曲线末端水平、变化小
- **未收敛**:曲线末端仍在漂移
- **振荡**:曲线周期性波动 — 可能是非定常流或网格不够细
- **多 case 对比**:看相同条件下不同 case 的曲线趋势

## 4. extract 标准输出(stdout)

`inp-tool post extract` 不写文件,直接打印到 stdout。每个 case 一段:

```
[Case: case_01]
  Ma = 9.887, H = 30.00 km, α = 0.000°, β = 0.000°
  Fx = 5.4916e+06 N, Fy = 9.5117e+06 N, Fz = -1.9460e+07 N
  Mx = 3.1770e+08 N·m, My = 1.2242e+09 N·m, Mz = 6.8474e+08 N·m
  D  = 5.4916e+06 N, L  = -1.9460e+07 N, L/D = -3.5439
  CD = 0.12345, CY = 0.21368, CL = -0.43742
  Cmx = 7.139e-01, Cmy = 2.750e+00, Cmz = 1.539e+00
  Q  = 3.217e+03 Pa, Re = 1.846e+06 /m, Xcp = -22.30 m
```

`--quiet` 抑制 case 详细信息,只保留进度行。

## 5. 重要约定

### 5.1 单位

| 量 | 单位 |
|---|---|
| 长度 | m |
| 力 | N |
| 力矩 | N·m |
| 压强 | Pa |
| 速度 | m/s |
| 温度 | K |
| 攻角 / 侧滑角 | **度**(不是弧度) |
| Reynolds | **/m**(单位长度) |

### 5.2 符号约定

- **Fx**:体轴 x 方向力(CFD++ 默认 x 沿来流方向)
- **D**:阻力(wind-axis x),`D = Fx·cos α + Fz·sin α`
- **L**:升力(wind-axis -z),`L = -Fx·sin α + Fz·cos α`
- **Cmx/Cmy/Cmz**:绕 (Xref, Yref, Zref) 的力矩系数
- **Xcp**:压心 X 坐标 — 升力作用点(若 Fz > 0)

### 5.3 数值精度

- Excel 保留浮点精度(openpyxl 默认 ~15 位)
- stdout 打印用 `%.4g` / `%.4e` / `%.5f`(可读优先)
- 收敛报告 CV 用 4 位小数

## 6. 数据持久化建议

| 用途 | 推荐 |
|---|---|
| 报告(给同事看)| Excel + png |
| 自动化流水线 | txt(grep-friendly)+ Python API |
| 论文图表 | png(150 DPI 矢量保留)或 Python API + 自定义 matplotlib |
| 长期归档 | 同时存 mcfd.info1 + ForceReport.xlsx + convergence_report.txt 一套 |
